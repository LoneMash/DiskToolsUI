# DiskToolsUI ⚡

Outil WPF/C#/PowerShell pour gestion disques HDD/SSD.

## Fonctionnalités
- Lecture numéros série disques
- Scripts PowerShell intégrés
- UI moderne Dark theme

## Installation
```powershell
git clone https://github.com/LoneMash/DiskToolsUI.git
cd DiskToolsUI
dotnet restore
dotnet run
